## [0.1-beta] — TBD

### Added
- Initial private beta release
- BRep solid and shell modeling primitives
- Boolean operations (union, cut, intersect)
- Fillet, chamfer, bevel
- Face extrude, profile extrude, revolve
- Loft, patch from curves, bridge, fill hole G0/G1
- 2D curve toolset (circle, arc, ellipse, rect, spline CV/EP, fillet 2D, offset, join, extend)
- Imprint and trim on shells
- Shift+click multi-select edge/face
- DWG/DXF import
- Full undo/redo (50-level snapshot)
