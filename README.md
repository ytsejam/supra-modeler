# Supra Modeler

**Professional NURBS/BRep direct modeling for Autodesk Maya.**

Supra Modeler brings Plasticity-style direct modeling workflows into Maya — built on OpenCASCADE Technology (OCCT) with a permanent viewport context, full undo history, and a clean BRep topology model.

Designed for automotive and visualization pipelines.

---

## Status

🔴 **Private Beta** — invitation only

## Features (Beta)

- BRep solid and shell modeling (box, cylinder, sphere, torus primitives)
- Boolean operations (union, cut, intersect)
- Fillet, chamfer, bevel
- Face extrude, profile extrude, revolve
- Loft, patch from curves, bridge, fill hole G0/G1
- 2D curve toolset: circle, arc, ellipse, rect, spline CV/EP, fillet 2D, offset, join, extend
- Imprint and trim on shells
- Shift+click multi-select edge/face
- DWG/DXF import
- Full undo/redo (50-level snapshot)

## Requirements

- Autodesk Maya 2025
- Windows 10/11 x64
